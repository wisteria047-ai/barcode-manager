# バーコード管理ツール

バーコードで物品を管理。CSV取込、ラベル印刷、スキャンで貸出/返却を記録。

## 主な機能

- CSV/Excelファイルから物品データを取込
- バーコード生成（Code128 / Code39 / QR）
- ラベル印刷（A4用紙、11種のプリセット、PDF出力）
- バーコードスキャン（手入力 / カメラ読取）
- 貸出・返却管理（ステータス自動変更、延滞表示）
- 操作履歴・Undo対応
- オフライン動作（インターネット不要）
- 日本語 / 英語 対応

## こんな方に

- 備品・機材の貸出管理をしたい
- 棚卸し作業を効率化したい
- ラベルシールを印刷して物品に貼りたい
- Excelの管理台帳をもっと便利にしたい
