# Barcode Manager

Manage items with barcodes. Import CSV, print labels, scan to track lending and returns.

## Key Features

- Import item data from CSV / Excel files
- Barcode generation (Code128 / Code39 / QR)
- Label printing (A4 paper, 11 presets, PDF output)
- Barcode scanning (manual input / camera)
- Lending & return management (auto status, overdue alerts)
- Operation history with Undo support
- Fully offline (no internet required)
- Japanese / English support

## Who Is This For?

- Equipment and asset lending management
- Inventory auditing and stocktaking
- Printing barcode labels for physical items
- Upgrading from spreadsheet-based tracking
